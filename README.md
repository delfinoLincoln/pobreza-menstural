# Projeto Pobreza Menstrual - IFPR Campus Paranaguá

## 📌 Sobre o Projeto
Este projeto foi desenvolvido para a disciplina de **Língua Portuguesa II** do **IFPR - Campus Paranaguá**, com o objetivo de conscientizar sobre a pobreza menstrual no Brasil.

**Acesse o site:** [pobreza-menstrual.vercel.app](https://pobreza-menstural.vercel.app)

## 🎯 Objetivos
- Conscientizar sobre a realidade da pobreza menstrual
- Informar sobre políticas públicas existentes
- Combater o tabu e a desinformação sobre menstruação
- Promover dignidade menstrual para todas as pessoas

## 🔍 O que é Pobreza Menstrual?
A pobreza menstrual é a falta de acesso a produtos de higiene menstrual, infraestrutura adequada (como água e saneamento) e informação sobre o assunto. Afeta milhões de pessoas que menstruam no Brasil e no mundo.
